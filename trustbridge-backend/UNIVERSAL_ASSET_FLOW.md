## üîÑ Complete Tokenization Flow

### **Phase 1: Asset Submission**

#### **Step 1: Asset Type Selection**
```
User ‚Üí Select Category ‚Üí Choose Asset Type ‚Üí Set Value Range
```

**Example Flow:**
1. User selects "FARM PRODUCE"
2. Chooses "Cocoa" from cash crops
3. Sets value range: $1,000 - $5,000,000
4. System validates against asset type configuration

#### **Step 2: Evidence Collection**
```
Upload Documents ‚Üí Verify Requirements ‚Üí Quality Check
```

**Required Documents by Asset Type:**

**Farm Produce (Cocoa):**
- Export Certificate
- Quality Grade Report
- Origin Proof
- Storage Facility Certificate

- Title Deed
- Survey Report
- Soil Analysis
- Water Rights Certificate

**Real Estate (House):**
- Title Deed
- Survey Report
- Valuation Report
- Insurance Policy

**Vehicles (Car):**
- Registration Document
- Insurance Certificate
- Inspection Report
- Service History
