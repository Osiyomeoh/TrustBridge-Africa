🔧 Technical Implementation

Profile Dashboard Flow:

1. Profile.tsx uses useAssetByOwner(address) hook
2. Hook calls apiService.getAssetsByOwner(owner)
3. API calls GET /api/assets/owner/${owner}
4. Backend calls assetsService.getAssetsByOwner(owner)
5. Returns assets from database where owner = address

Admin Assets Flow:

1. AdminAssets.tsx fetches from /api/assets
2. Backend calls assetsService.getAssets()
3. Returns all assets from database
4. Admin can approve/reject using /api/admin/approve-asset
5. Admin service updates asset status and creates Hedera tokens
