🧪 How to Test

1. Create RWA Assets First

1. Go to your profile or asset creation flow
2. Create some RWA assets (they get stored in the Asset database)
3. These assets will have status 'PENDING' initially

2. Access Admin Assets Page

1. Go to: http://localhost:3000/admin/assets
2. Make sure you're logged in with admin privileges
3. You should now see the real RWA assets you created

3. Test Approval Process

1. Click on any PENDING asset to view details
2. Click "Approve" button
3. Asset status should change to 'ACTIVE'
4. Check console for "Asset approved successfully"

4. Check Browser Console

Look for these log messages:
- "Fetching RWA assets from backend..."
- "Assets fetched: {data: [...]}"
- "Transformed assets: [...]"
- "Approving asset: [assetId]"
- "Asset approved successfully"
